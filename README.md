# GHDropMenuDemo
**自由定制电商筛选菜单**

**超级简单好用的电商筛选菜单** 
<br/>项目截图<br/>
![image.png](https://upload-images.jianshu.io/upload_images/1419035-cf8d52506302fea5.png?imageMogr2/auto-orient/strip%7CimageView2/10/w/240)

* **自定义筛选标题,自定义筛选内容,自定义筛选标签,自定义筛选头部内容**
* **数组越界处理**
* **价格输入筛选**
* **实现tag标签,单选,多选,取消选中效果**
* **保留上次选中选项**
* **动画展开,移除**
* **可以重新传入模型,重新刷新数据源**
* **选中内容通过代理的方式回调**
* **对原有项目无污染,直接拖进项目即可使用**

* 导入
